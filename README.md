# Android Lab 13 - Integracion de APIs REST

Laboratorio Android sobre consumo de servicios REST con Retrofit. Manejo asincrono de datos, serializacion JSON y actualizacion de la UI con resultados de red.

## Temas cubiertos
- Cliente HTTP con Retrofit 2
- Serializacion JSON con Gson o Moshi
- Coroutines para llamadas asincronas
- Manejo de estados: loading, success, error
- RecyclerView con datos de la API
